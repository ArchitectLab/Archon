# Politique de securite d'Archon

La securite est une fondation, pas une etape. Merci de nous aider a garder Archon sur.

## Signaler une vulnerabilite
- **N'ouvre pas d'issue publique.**
- Ecris a **security@joinarchitectlab.com** *(adresse a confirmer)*, ou utilise les
  **GitHub Security Advisories** (onglet "Security" du depot > "Report a vulnerability").
- Donne le maximum de details : version concernee, etapes de reproduction, impact estime.

Nous accusons reception sous quelques jours ouvres et te tenons informe de la correction.
Nous pratiquons la **divulgation coordonnee** : on convient ensemble d'une date de
publication une fois le correctif disponible.

Les aspects securite sont suivis avec les ingenieurs de **Bunkerity** (editeurs du WAF
open source BunkerWeb).
