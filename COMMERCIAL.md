# Licences commerciales

Archon est publie en open source sous **AGPL-3.0-or-later**. L'AGPL impose notamment
que toute version modifiee mise a disposition via un reseau (SaaS) publie son code
source aux utilisateurs du service.

Si ces obligations ne conviennent pas a ton usage, par exemple :
- integration d'Archon dans un produit ferme (proprietaire) ;
- hebergement en service en ligne sans publier tes modifications ;
- redistribution sous une licence differente de l'AGPL ;

alors une **licence commerciale** est disponible.

- **Co-detenteurs des droits** : Architect Lab, Morphaius et Bunkerity.
- **Contact licences** : contact@morphaius.com

> Ce modele de double licence repose sur le `CLA.md` : chaque contribution est couverte
> par le CLA, afin que les co-detenteurs puissent relicencier le code (y compris a titre
> commercial). La repartition entre co-detenteurs releve d'une convention de
> co-propriete a etablir entre les trois parties.
