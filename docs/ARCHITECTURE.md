# Architecture d'Archon  (a co-ecrire)

Esquisse a remplir ensemble. Quelques briques pressenties :

- **Coeur** minimal : orchestrateur, bus d'evenements, gestion des plugins, securite.
- **Couche IA** : comprehension du langage naturel, intentions, automatisations.
- **Plugins** : interface stable, isolation, permissions explicites par plugin.
- **Frontends** : dashboards qui se reconfigurent (genere/decrit a la volee).
- **Securite** : cloisonnement, chiffrement, journalisation, conception avec Bunkerity.
- **Deploiement** : 100 % local, hybride, ou cloud (au choix de l'utilisateur).

> Principe directeur : la confiance ne se decrete pas, elle se verifie.
