Read the `readme.md` file within this skill, and explore the other available files (tokens, components, foundations, ui_kits, assets).

If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. If working on production code, you can copy assets and read the rules here to become an expert in designing with this brand.

Core identity in one breath: the **architect's blueprint** (48px dark grid on Deep Void `#0B1220`) crossed with the **Morphaius spectrum** (green `#2ECC9A` → blue `#2A6BF2` → violet `#8A2BE2` → magenta `#E612D9`). Keep the spectrum under ~10% of any composition — it lives on strokes, rules, the logo and clipped text, never as a flat fill. Type: Space Grotesk (titles), Inter (body), JetBrains Mono (tags/code, always uppercase). Voice: the engineer-builder — terminal-flavoured, verbs first, the trailing magenta `_` cursor. No emoji.

If the user invokes this skill without any other guidance, ask them what they want to build or design, ask some questions, and act as an expert designer who outputs HTML artifacts _or_ production code, depending on the need.
